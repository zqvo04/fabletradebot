# EXPERIMENTS — 설계 구간 반증 로그 (전부 기록: 성공·실패 포함)

설계 구간: **2023-06-01 ~ 2026-01-31** (탐색·튜닝은 여기서만).
홀드아웃: 2026-02-01 ~ 2026-07-10, 1회만 (VALIDATION.md).
측정 모드 = 포트폴리오 캡 해제 + 단일 티어(순수 신호 엣지 분리 목적).
모든 수치는 수수료(테이커 0.05%/side)·슬리피지·펀딩 드래그 반영 후.

## E1 — 초안 신호군 기준선 (S1 되돌림 / S2 스퀴즈 / S3 스윕, 롱숏)
- 제약 포함: n=242, **avg −0.16R**, 승률 40.5%. 무제약: n=287, −0.15R.
- 확신도 5분위 비단조(고확신이 오히려 나쁜 구간 존재). **기준선 자체가 음수.**

## E5 — 손실 원인 분해 (스펙트럼 653 트레이드)
- 손절폭 <1%: −0.37R / 1~2%: −0.14R / 2~4%: −0.12R / ≥4%: **+0.12R**
  → 1H 노이즈 밴드 안의 손절은 스윕당함 + 비용/R 비율 악화.
- BTC 방향 일치: −0.08R vs 불일치 −0.27R.
- 조치: SL 변동성 하한 2×ATR 도입, S1에 BTC 방향 하드 필터 → **재측정 결과
  −0.14→−0.08R 한계 개선에 그침. 정적 슬라이스 효과는 교란이었음(정직 기각).**

## E6 — 조건부 포워드 수익률 스터디 (신호 원재료 탐색)
72h 포워드, 12자산 가중평균, 왕복비용 ~0.2% 기준:

- 첫 크로싱(에피소드) 기준 재측정: RSI<20 +0.84%/+2.14% — 여전히 양수.

## E7 — 신규 신호 CAPREV/BRK 트레이드 시뮬레이션
- CAPREV(RSI<20 롱): **−0.29R** (n=275). 손절·확인봉·홀드 구조 5개 변형 전부 음수
  (−0.10~−0.42R).
- BRK(정렬 7d 돌파 롱): +0.05R, conf 상위분위 +0.15R. 종료 구조가 문제로 의심됨.

## E8 — CAPREV 부검 → 기각
홀드-72h/6ATR 손절 구성에서: 타임아웃 생존 트레이드 평균 **+3.53%**(스터디와 일치)
그러나 **40%가 −9%(6ATR) 손절을 먼저 맞음**(평균 −7.7%). 즉 포워드 +드리프트는
재앙 분산 경로 위의 평균이며, 생존 원칙(고정 리스크+손절+청산 봉쇄)과 양립 불가.
**CAPREV 기각. cap_enabled=False 플래그로만 잔존.**

## E9 — BRK 종료 구조 A/B (승자 = V1 기본값)
| 구조 | 전반부 | 후반부 |
|---|---|---|
| TP1 1.5R+트레일3+72h컷 (초안) | +0.24R | −0.07R |
| 트레일 없음, 4H bias 플립만 | +3.06R | +0.52R |
| 트레일8 + bias 플립 | +1.58R | +0.02R |
| 트레일12 + bias 플립 | +2.04R | +0.32R |
| **트레일8 (버그 수정: 진입부터 활성)** | **+1.30R** | **+0.34R** |
- 부분 TP·시간손절은 추세 승자를 잘라 기대값 파괴 → 제거.
- 제약 포함 3후보 비교: 트레일8이 MDD 최저(−24.6%)·후반부 최고(+0.34R)·표본 최다
  → **채택: 트레일8 샹들리에 + 4H bias 플립 청산, TP/타임아웃 없음.**
- 엔진 버그 발견·수정: 트레일이 TP1 이후에만 활성이던 문제(테스트 추가).

## E9b — 확신도 예측력 (최종 구조)
- 스펙트럼 n=310: corr(conf, R)=0.005, 5분위 비단조. conf≥0.6/0.7/0.8 컷 모두
  기대값 개선 없음. **확신도 티어 사이징 기각 → 균일 0.6% 리스크, 단일 티어.**
  (전방 채점이 conf 예측력을 계속 측정 — 근거 생기면 재도입.)
- RANGE 국면 돌파 +1.28R vs TREND 중 +0.38R (신선한 추세 시작이 최고).
- 자산별: 대형주 전부 양수, ONDO/TAO/WLD/SUI 음수 — 사후 제외는 과적합 위험이라
  유니버스 유지(자산 캡 3x가 노출 제한). 전방에서 재평가.

## E10 — 공격 레이어 (V1.1, 사용자 지시: BTC 단독 백테스트)

### E10a — "좋은 기회" 예측자 탐색 → 기각
- RANGE→돌파(신선 추세) 판별자: 유니버스 H1 +4.08R / **H2 −0.13R로 반전**, BTC
  단독에선 분리 없음(0.70 vs 0.84) → 기각. 확신도는 이미 E9b에서 기각.
- **결론: 어느 트레이드가 대박일지 사전 예측은 이 데이터에서 불가.** 공격성은
  예측이 아니라 사후 증거에 조건화한 기계적 정책으로만.

### E10b — 기계적 공격 정책 (BTC 단독, 설계 구간, 사전 표준값·격자탐색 없음)
| 구성 | 총수익 | MDD | MC 95%p MDD |
|---|---|---|---|
| P0 베이스 | +19.5% | −8.4% | −9.8% |
| P1 피라미딩(+2R/+4R, 최대 2) | +24.4% | −12.0% | −15.0% |
| P2 에쿼티 부스트(고점 2% 이내 ×1.5) | +29.7% | −12.4% | −14.4% |

- 반쪽 분할: H1 +32.7% / H2 +2.6% (양쪽 비음수).
- 사전값 ±20% 섭동(trigger 1.6/2.4R, boost 1.2/1.8×, max 1/3): 전부 양수 유지.
- 비용 2배: +21.0% 유지.

### E10c — 유니버스 전체 공격은 생존 게이트 붕괴 → BTC 한정 롤아웃
- 전 유니버스에 P3 적용: MC 95%p MDD **−53%**, P(>30%)=60%, 총수익도 146→100%로
  악화(알트 챕에서 피라미딩 = 2R 고점 추격). → `aggression_syms=("BTC",)`.
- BTC 한정 + 베이스 r 0.6→0.55% 축소: 유니버스 +160%, MC 95%p −27.0%,
  P(MDD>50%)=0 — 생존 게이트 통과. BTC 단독 +33.9%/MDD −16.2%.
- 홀드아웃 재개봉 안 함(신호 불변·사이징만 변경, MC로 검증) — 봉인 유지.
