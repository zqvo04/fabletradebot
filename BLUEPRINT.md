# BLUEPRINT — V1 Discrete 방향성 레버리지 진입 엔진

> 한 줄: 코인별 독립 스캔 → 컨플루언스 확신도(0~1) → 자격 있는 순간에만 discrete
> 진입. 손절이 청산보다 항상 먼저 맞도록 구조적으로 봉쇄한 위에서 기하수익을
> 최대화한다. 성패 판정은 백테스트가 아니라 페이퍼 전방 트랙.

> **개정 이력**: 초안의 신호군(S1 되돌림/S2 스퀴즈/S3 스윕, 롱숏 양방향,
> 확신도 4티어 0.5~1.5%)은 설계 구간 측정에서 반증되어 폐기·교체됐다.
> 전 과정은 EXPERIMENTS.md에 기록. 현재 문서는 측정 후 최종 설계다.

## 0. 성공 기준 (이원화)

| 기준 | 내용 | 판정 |
|---|---|---|
| 생존 (hard) | 어떤 경로에서도 청산 없음, MC 부트스트랩 95%p MDD ≤ 30%, P(MDD>50%) < 1% | 배포 게이트 |
| 수익 (aspiration) | 페이퍼 전방 월 기하수익 최대화, 최고의 달 +50~100% 영역 | 전방 트랙 |

전방 판정: **페이퍼 4주** 최소. 전방 샤프가 백테스트 샤프의 절반 미만이면 라이브 금지.

## 1. 정직성 선언 — 레버리지와 수익의 실제 관계

트레이드당 리스크를 고정(r)하면 수학적으로:

```
명목(notional) = equity × r / stop_frac        (손절폭이 결정)
증거금(margin) = notional / L                   (레버리지가 결정)
손절 시 손실   = equity × r                     (상수 — 레버리지와 무관)
```

즉 **레버리지 L은 손익 크기를 직접 키우지 않는다.** L이 하는 일은 (a) 같은 명목을
더 적은 증거금으로 여는 것(동시 포지션 여력), (b) 청산가 거리를 좁히는 것(위험).
수익을 키우는 실제 레버는 **확신도 티어별 리스크(r: 0.5→1.5%)와 타이트한 구조적
손절(명목 = r/stop_frac 배 확대)**이다. r=1.5%, stop=1.2%면 명목은 계좌의 1.25배.
"큰 달"은 10x 명목 몰빵이 아니라 고확신 트레이드 다수의 +2~3R 복리에서 나온다.
이것이 브리프 §5의 수식이 실제로 의미하는 바이고, 이 문서는 그 사실을 숨기지 않는다.

L 티어(2/3/5/10x)는 **명목 상한(equity×L)** 겸 **증거금 배율**로 쓰며, 청산 안전
규칙이 L의 물리적 상한을 만든다(§5).

## 4. 국면 인식 (1D 광역 + 히스테리시스)

BTC 1D 기준 광역 국면 + 자산별 로컬 보정.

피처(전부 t 마감 1D 봉 기준):
- `trend_dir`: close vs EMA100, EMA20−EMA50 부호
- `vol_pct`: 실현변동성(20D)의 1년 백분위
- `crash`: 5D 수익률 < −12% 또는 1D < −7%

상태 및 판정(우선순위 순):
1. **CRISIS**: crash 또는 (vol_pct>90 & 5D<−8%) → 신규 진입 금지, 기존 포지션 강제 점검
2. **HIGH_VOL**: vol_pct > 80 → L캡 3x, 역추세 신호 금지
3. **TREND** (UP/DOWN): |EMA20−EMA50|/ATR > 0.5 & close가 EMA100 같은 쪽 → L캡 10x, 추세 플레이북
4. **RANGE**: 그 외 → L캡 5x, 역추세(S3) 허용·추세추종 감점

히스테리시스: 상태 전환은 새 상태가 **2연속 1D 봉** 확인될 때만(CRISIS는 즉시 진입,
해제는 3봉). 상관 급등 경보: 유니버스 1H 수익률 30D 평균 페어와이즈 상관 > 0.8 →
신규 리스크 반감 + 동시 포지션 2개 제한.

## 5. 신호군 (측정 후 생존자)

각 1H 마감마다 코인별 독립 평가. 초안 4군 중 측정 생존자만 남았다
(반증 상세는 EXPERIMENTS.md E1~E9).

### BRK — 정렬 추세 브레이크아웃 롱 (주력, 유일한 생존 신호)
- 트리거: 1H 마감가 > 직전 7일(168봉) 최고가
- 필터: 1D bias=up & 4H bias=up & BTC 1D 방향=up & 거래량 ≥ 1.2× 중앙값(48)
  & 국면 ∈ {TREND, RANGE} (RANGE→돌파가 신선한 추세 시작으로 최고 성과)
- SL: max(돌파선 − 0.6×ATR1H, 종가 − 2.0×ATR1H 하한) — 변동성 하한 필수(E5)
- 강도(base): 돌파 마진, 사전 압축(BB폭 백분위), 돌파 봉 실체, 거래량 배율

### 펀딩 수정자 (가/감점, 단독 진입 아님)
- 8h 펀딩 90D z-score. 숏 크라우딩(z<−1.5)에서 롱 → +0.05 (스퀴즈 연료),
  롱 크라우딩(z>+1.5)에서 롱 → −0.10. OKX 펀딩 히스토리가 ~3개월뿐이라
  설계 구간 검증 불가 — **미검증·유계(±0.1)** 상태로 두고 전방 채점으로 판정.

### 기각된 신호 (측정 근거 — 재도입 금지 아님, 재측정 필요)
- 되돌림 재개(S1): 최적 변형도 −0.08R. 스윕 반전(S3): −0.12R. 스퀴즈 필터(S2): 표본
  부족·부호 불안정. **숏 전 계열**: 최근 16개월 12/12 자산에서 숏 방향 −3.1%/72h
  (숏스퀴즈 지배) — V1은 **롱 전용**, 약세 국면에서는 구조적으로 관망.
- 과매도 반등(CAPREV): 포워드 드리프트는 +지만 40% 빈도의 −9% MAE 경로 위에 얹혀
  있어 생존 호환 손절로는 포착 불가(E8) — 코드에 플래그로만 잔존.

### 확신도 결합
```
conf = 0.55×base + 0.25×국면적합 + 0.20×TF정렬 + funding_mod, clip[0,1]
```
- 진입 임계 conf ≥ 0.55. 셋업 없으면 **진입 안 함이 기본값** (관망도 전략).
- 동일 코인 복수 신호 동시 발화 시 conf 최고 1개만.

## 6. 확신도 → 레버리지/리스크 매핑 (측정 후 단순화)

**측정 결과 확신도는 이 신호의 R을 랭킹하지 못했다**(corr 0.005, 5분위 비단조 —
E9). 따라서 V1은 티어 사이징을 접고 균일 리스크로 간다. 티어 배관은 코드에
남아 있고, 전방 채점이 conf 예측력을 계속 측정해 근거가 생기면 재도입한다.

```
conf < 0.55 : 진입 안 함
conf ≥ 0.55 : L티어 5x, r = 0.6%   (단일 티어)
```
r=0.6%는 MC 생존 게이트(G7: 95%p MDD ≤ 30%)를 통과하는 최대 사이즈로 결정.

```
L_final = min(L_conf, L_regime, L_liq, L_asset)
L_liq   = floor_tier( 1 / (3×stop_frac + mmr_buffer) ),  mmr_buffer = 1.5%
```

**청산 안전 절대 규칙**: 격리 마진에서 청산까지 가격 거리 ≈ (1/L − mmr). L_liq는
**청산 거리 ≥ 3×손절 거리**를 강제한다. 예: stop 2% → L_liq ≤ 1/(0.06+0.015)=13.3
→ 10x 허용. stop 4% → 7.4 → 5x. stop 8% → 3.9 → 3x. 손절이 항상 먼저 맞는다.
포지션 오픈 시 `liq_price`를 계산·기록하고, 엔진이 매 봉 `SL이 liq보다 가깝다`를
assert(위반 시 즉시 에러 — 테스트로 증명).

사이징:
```
risk_amt = equity × r × dd_governor_mult × corr_mult
notional = min(risk_amt / stop_frac, equity × L_final)
margin   = notional / L_final
(notional 캡에 걸리면 실질 리스크 < r — 그대로 둔다, 확대 금지)
```

## 7. 포트폴리오 리스크 골격

| 장치 | 규칙 |
|---|---|
| 코인당 포지션 | 1개 (반대 신호 = 청산 후 다음 봉 재평가) |
| 동시 포지션 | ≤ 4개 (상관 경보 시 ≤ 2) |
| 총 오픈 리스크 | ≤ 4.5% equity (신규 진입이 초과시키면 스킵) |
| 총 증거금 | ≤ 60% equity |
| DD 거버너 | 고점 대비 −10% → r 반감, −20% → 신규 정지(−15% 회복 시 해제) |
| 서킷 브레이커 | 실현손실 24h 내 ≥ 4% equity → 24h 신규 정지 |
| CRISIS 국면 | 신규 금지 + 오픈 포지션 트레일 타이트닝 |

## 8. 포지션 수명주기 (측정으로 선택 — E9)

- **진입**: conf ≥ 0.55 & 리스크 게이트 전부 통과. 다음 1H 봉 시가 진입(보수적).
- **손절**: 구조적 무효화점(§5), 진입 후 타이트닝만 허용.
- **익절 없음**: 부분 TP·시간손절은 이 신호의 추세 승자를 잘라 기대값을 파괴함을
  측정으로 확인(TP1.5R+72h컷: −0.07R vs 제거: +0.34R, 최근 반쪽). 청산은
  **8×ATR1H 샹들리에 트레일**(진입 직후부터)로만.
- **국면 청산**: CRISIS 진입 → 즉시 청산. 4H bias 하향 반전 2봉 확인 → 청산.

## 9. 비용 모델 (정직 반영)

- 수수료: 테이커 0.05%/side (진입·청산 각각)
- 슬리피지: 자산군별 0.02~0.08%/side (§2 표)
- 펀딩: 보유 중 8h 경계마다 실제 펀딩률 × 명목/equity 부호 반영
- 스트레스 게이트: 전 비용 2배에서도 기대값 > 0 유지해야 통과
